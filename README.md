# 황금거위 리포트

GitHub Pages 배포를 염두에 두고 정리한 황금거위 광고 성과/시뮬레이션 리포트입니다.

## 파일 구성

- `index.html`: 리포트 허브 시작 페이지
- `ad_revenue_performance_section.html`: 광고 매출 및 성과 보고서
- `ad_revenue_simulation.html`: 공격적 집행 기준 매출 시뮬레이션

## 배포 방법

1. 이 폴더를 GitHub 저장소로 올립니다.
2. GitHub Pages를 `main` 브랜치 루트로 설정합니다.
3. 배포 후 `index.html`을 시작 페이지로 사용합니다.
